#contributer

## Description

This is a basic JavaScript project that allows users to:

* Search for weather information by city
* Search for movies using an API

The application fetches real-time data from external APIs and displays it in a simple user interface.

## Features
Get current weather by city name
 Search movies by title
 API data fetching using JavaScript
 Simple and responsive UI
 Error handling for invalid inputs
## Tech Stack
* HTML
* CSS
* JavaScript (Vanilla JS)
* REST APIs

## APIs Used

* Weather API(weatherapi.com) 
* Movie API (www.omdbapi.com)

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/PRIYANSHUKUMAWAT1906/Javascript-basic-poject
```
2. Open the project 
3. Open `index.html` in your browser
